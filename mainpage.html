Magnetic Storage

Magnetic storage store information using the presence and absence of magnetism which represent binary data. They contain disks which are called platter and on each platter, it is divided into sectors which is further dividied into tracks where it stores data. The read/write head then moves across the radius detecting magnetised and demagnetised spots and send data via wire.

Advantages

  • Large number of read/write cycles
  • Relatively Cheap
Disadvantages
  • Power consuming as it contains moving parts
  • Louder than SSD
  • Less Durable from having many moving parts

TCP/IP Model

The TCP/IP model refers to a procedure and protocols used in order to transmit file and establish connection between computer device. It consists of 4 layers
  1. Application layer
  2. Transport Layer
  3. Network Layer
  4. Link Layer
